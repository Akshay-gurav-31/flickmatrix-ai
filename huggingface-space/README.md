# FlickMatrix AI

A production-ready **Movie Recommendation System** built on the MovieLens Latest-Small dataset (100K ratings, 9,000 movies, 600 users).

**Features:**
- 5 trained ML models (Popularity, Content-Based, Item-CF, SVD, Hybrid)
- Explainable AI recommendations with reasoning
- Movie search and similarity discovery
- Performance benchmark analytics
